关于对自治区政协十二届二次会议第0041号提案的答复
发布日期：2019-08-20 00:00:00　|　 文章来源：内蒙古自治区财政厅

内财办建提〔2019〕261号

杨富有委员：

您提出的《关于民族教育补短板促发展的提案》收悉。经研究，现就财政相关事宜答复如下：

自治区党委和政府历来高度重视民族教育发展。为支持我区民族教育发展，自治区财政在积极争取中央财政支持的同时，不断加大对民族教育的投入，在资金分配上尽可能对少数民族旗县给予倾斜。

2011年—2018年，中央财政累计支持我区少数民族地区教育特殊补助资金1.94亿元。同时，自治区各级财政克服财力困难，努力加大对民族教育的投入，支持民族教育发展的各类专项资金由“十一五”时期的近12亿元，提高到“十二五”时期的近20亿元，初步形成了支持民族教育发展的经费保障体系和工作机制。一是为解决民族中小学校公用经费不足的问题，自治区人民政府办公厅印发了《进一步完善城乡义务教育经费保障机制实施方案的通知》（内政办发〔2016〕32号），对民族学校按照双语授课学生人数和年生均公用经费基准定额标准的10%增加公用经费补助。二是除每年拨付正常公用经费外，自治区财政每年安排民族教育专项资金从2007年的2000万元增加到目前的5700万元，各级财政也相应安排了专项资金，累计年投入5000多万元。三是自2007年起加大了对蒙汉双语教学义务教育和普通高中学校寄宿制学生的生活费补助力度，现标准为每人每年小学生1350元、初中生1620元、高中生1890元，除中央补助资金外，剩余资金全部由自治区本级财政负担。这些政策措施的实施，有效地促进了我区民族教育的发展。

近年来，自治区经济增长明显放缓，财政收支矛盾十分突出，按照现行财力保障和支出责任相匹配的财政管理体制以及《内蒙古自治区人民政府贯彻国务院〈关于基础教育改革与发展决定〉的实施意见》（内政发〔2002〕25号）确定的“基础教育实行在自治区人民政府领导下，由各级政府负责、分级管理、以旗（县）为主”的管理体制，建议盟市、旗县政府进一步调整支出结构，努力加大教育投入，多渠道筹措资金解决民族教育发展问题。

感谢您对财政工作的支持，欢迎多提宝贵建议。
