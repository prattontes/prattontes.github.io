李文亮于2020年2月6日晚在重症监护室去世

2020年02月06日 23:08:10
来源：界面新闻

因公布武汉新型冠状病毒肺炎疫情被训诫的武汉市中心医院眼科医生李文亮，感染新型冠状病毒肺炎后病情加重，于2020年2月6日晚去世。

据生命时报报道，李文亮医生因感染新型冠状病毒肺炎，于2月6日21点30分病逝。界面新闻从李文亮同学处获悉，他的病情一度好转，但近几日忽然加重。

李文亮，籍贯辽宁省锦州市北镇市，今年34岁。2004年李文亮参加高考并报考武汉大学临床医学七年制专业，在厦门短暂工作3年后，重返武汉在武汉市中心医院工作至今。

武汉新型冠状病毒肺炎疫情爆发后，李文亮被誉为此次疫情“吹哨人”。2019年12月30日下午，身为武汉市中心医院眼科医生的李文亮，在同学群中发布了一条关于华南海鲜市场出现不明原因肺炎的信息。2020年1月3日，他因“在互联网上发布不实言论”而被辖区派出所提出警示和训诫。

据中国经营报此前报道，1月8日，李文亮在接诊时遇到不明原因的病毒性肺炎患者并受到感染。1月10日，李文亮出现咳嗽发热等症状，随后病情变得严重。2月1日，李文亮确诊为新型冠状病毒。
